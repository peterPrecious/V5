Options de réglage de votre navigateur Web


Afin d’utiliser notre service sur Internet, vous devez régler certaines options de votre navigateur Web ou des options des barres à outils de programmes spécialisés tels que Google ou Yahoo que vous aurez installées. Si vous éprouvez des difficultés à accéder aux cours, c’est généralement parce qu’un programme tiers empêche le fonctionnement de votre navigateur web. Typiquement les fenêtres pop-ups dont vous avez besoin dans le programme de formation.
Commencez par démarrer notre Browser Test en cliquant sur le bouton ci-dessous, et retourner à cette page afin de lire les instructions indiquant comment changer vos options de réglages si cela est nécessaire.
Si aucun paramètre de « Browser », « Pop-up Blocker », « Cookies » ou « Flash » n'indique « Passed » veuillez faire défiler le menu déroulant pour obtenir des renseignements sur la façon d’ajuster vos options de réglages et accéder au système.

Pop-Up Blockers

Si les résultats de votre Test Browser indiquent que vous avez un pop-up blocker problem assurez-vous que vous avez désactivé tous les pop-up blockers (ou pubs bloquées) sur votre ordinateur, et redémarrez le Browser Test afin de voir si vous éprouvez encore des difficultés.

D’où viennent les pop-up blockers? Très souvent, ils proviennent de la barre à outils personnalisée que les fournisseurs de services Web tels que Google, Yahoo et AOL vous autorisent à télécharger et installer dans votre navigateur. Vous ne réalisez peut-être pas que vous avez un pop-up blocker activé sur une de ces barres à outils. Le pop-up blocker apparaît comme un bouton sur la barre à outils qui peut être activé ou désactivé en cliquant dessus. Si le bouton vous permet de « Allow all pop-ups fom vubiz.com », alors faites-le. Si un bouton n’apparaît pas sur la barre à outils, vous pourrez désactiver le pop-up blocker en ouvrant votre navigateur; Cliquer sur « View », ensuite cliquer sur « Toolbars » à ce point-ci vous aurez à cliquer sur « Web Features », cliquer ensuite sur le nom de la barre à outils que vous voulez désactiver.

Vous devez avoir plus d'une source de pop-up blockers. Le navigateur Web Internet Explorer lui-même et la plupart des programmes de sécurité et/ou anti virus contiennent aussi des pop-up blockers et ils doivent être désactivés. Assurez-vous que vous avez désactivé TOUS les pop-up blockers et démarrez à nouveau le Browser Test pour savoir si vous éprouvez encore des difficultés.

Comment désactive-t-on les pup-up blockers? Ceci varie selon la source du pop-up blocker.


Vous pouvez désactiver le pop-up blocker intégré dans votre navigateur Internet Explorer dans le menu « Tools ». Vérifiez que les autres pop-up blockers qui devraient être désactivés dans les programmes suivants le sont :

• Google Toolbar
• Yahoo Toolbar
• Norton Internet Security or Anti Spam
• McAfee
• Ad-Aware
• AOL

Autres réglages du navigateur : Caching Pages


Parce que les pages de ce service changent chaque fois que vous visitez Vubiz, vous devez vous assurer que votre navigateur ne fait pas qu’afficher la version précédente d’une page, i.e. une page « cache ». Pour confirmer que vous lisez de nouvelles pages de notre site, aller au menu « Tools » d'Internet Explorer; cliquer sur « Internet Options »; « Browsing History », cliquer sur « Setting ». Assurez-vous que vous choisissez « Every time I visit the webpage ».

Autres réglages du navigateur : Sites de confiance


Si vous avez essayé les étapes ci-dessus et éprouvez toujours des difficultés, veuillez essayer d’ajouter « Vubiz.com » comme « trusted site » (site de confiance) dans vos réglages de sécurité. Voici la démarche à suivre :

• Aller au menu « Tools » d’Internet Explorer
• Cliquer sur « Internet Options »
• Sélectionner l’onglet « Security »
• Cliquer sur l’icône « Trusted sites »
• Cliquer ensuite sur le bouton « Sites » en bas à droite de l’icône « Trusted sites »
• Ajouter « *.vubiz.com » dans « Add this website to the zone » champ
• Décocher « Require server verification (https:) for all sites in this zone »
• Cliquer sur le bouton « Add »
• Cliquer sur le bouton « Close »
• Cliquer sur le bouton « OK »
• Fermer Internet Explorer, et redémarrer.

Autres réglages du navigateur : Réglages de sécurité


Il est également possible que vos normes de réglages de sécurité soient très élevées pour que votre navigateur puisse fonctionner adéquatement sur notre système. Si votre Flash Player semble fonctionner adéquatement, et si vous êtes sûr que vous avez désactivé tous les pop-up blockers de votre système et pourtant éprouvez toujours des difficultés, veuillez réinitialiser vos réglages de sécurité par défaut.
